# Dodge – Design

## Goal
Rebuild programming fluency while learning effective human–AI collaboration by building small, shippable games.

## Platform
Python + Pygame Zero (desktop)

## Core Loop
Player moves to avoid falling obstacles. Survival time equals score.

## MVP
- Player can move on screen
- Obstacles fall
- Collision ends game

## Non-goals (for now)
- Menus
- Animation
- Sound
- High scores
